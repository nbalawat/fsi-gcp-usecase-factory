Option B uses the GEOGRAPHIC MAP as the page spine. The home surface renders a 2x2 US census-region tile grid (Northeast / Midwest / South / West), each tile colored by aggregate watchlist density (5-step heat scale, all from tokens — no arbitrary values). Below the tile grid, a state-cluster strip lets the user drill from "where is the risk?" to "which facilities are driving it?" without leaving the metaphor.
